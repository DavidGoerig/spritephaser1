## FORMAT DE CHAQUE ENTRÉE

```
ID | Nom | Espèce | Élément1 | Élément2 | Classe | Faction | Région | Arc(s) | Événements | Rôle narratif
```

**Légende Rôle narratif :**
- `[C]` = Cause d'un événement (ou complice direct)
- `[V]` = Victime d'un événement
- `[H]` = Héros/Agent qui intervient dans un événement
- `[T]` = Témoin (vivant)
- `[P]` = Profiteur (bénéficiaire involontaire ou délibéré)
- `[F]` = Fantôme (mort mais présent dans le lore d'autres)
- `[PIVOT]` = Connecte deux arcs

## NOTES POUR LA RÉDACTION DES LORES

### Règles de cohérence des noms
- **Pyriens** : noms courts, consonnes dures (Brasia, Cendrix, Gravel, Thryss)
- **Néréides** : noms fluides, terminaisons en -ara, -el, -eth (Lysara, Thariel, Crysta)
- **Aérides** : noms légers, consonnes doubles (Talvyn, Serathis, Yrith)
- **Sylvans** : noms naturels, terminaisons en -ae, -orn, -eth (Sylvae, Kessyn, Thalorn)
- **Chthoniens** : noms graves, consonnes profondes (Drevaya, Karm, Thyss)
- **Umbriens** : noms ambigus, souvent courts avec consonnes glissantes (Vethis, Nyrreth, Shael)
- **Givrins** : noms froids, terminaisons en -ra, -en, -ris (Sylkra, Aethren, Thyval)
- **Hommes-Liens** : aucune convention — reflet de leur diversité

### Âges des personnages à An 0 (Aujourd'hui)
- Vieux (60-120 ans) : témoins directs des événements anciens (-87 à -64)
- Adultes (30-60 ans) : ont grandi dans les conséquences des événements
- Jeunes (20-30 ans) : les 39 de Virex + génération post-Saturation
- **Note :** certains personnages Givrin peuvent être très vieux (stase culturelle)

### Distribution des factions
Viser pour chaque arc :
- ~30% Dissidentes (résistance, personnages les plus riches narrativement)
- ~25% Empire (agents, administrateurs, croyants sincères)
- ~20% Indépendants (ni blanc ni noir — souvent les pivots)
- ~15% Convergence (croyants légitimes + fanatiques)
- ~10% Confédération / Partis locaux (acteurs régionaux)
